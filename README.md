# Basic Calculator (Python)
A simple python program for learning purposes that performs basic arithmetic operations: addition, subtraction, multiplication and division. 

This project is designed for beginners learning Python and demonstrates basics concepts such as : 
. Output function
. User input
. Conditional statements
. Variables and types of variable

# How it works
1. The program asks the user the first number
2. The user enters the first number
3. The program asks now the second number
4. The user enters again the second number
5. The program asks the user the operation (+, -, *, /)
6. The user should enter the operation 
7. The selected operaton is done between both numbers by the program and prints the result


# Requirements
. Python 3.x

# How to run 
git clone git@github.com:hermesged/basic-calculator-python.git
cd basic-calculator-python
python calculator.py
